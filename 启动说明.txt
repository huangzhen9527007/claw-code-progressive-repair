Claude Code TUI 启动说明
============================================

一、快速启动步骤：

1. 双击运行 "TUI启动器.bat"
2. 如果首次运行，程序会自动检查并安装依赖
3. 按照提示配置API

二、API配置选项：

选项1: 使用本地Ollama（推荐）
--------------------------------
1. 安装 Ollama: https://ollama.com/
2. 下载模型: ollama pull llama3.2
3. 启动 Ollama 服务
4. 确保 .env 文件中使用Ollama配置

选项2: 使用DeepSeek API
--------------------------------
1. 注册 DeepSeek 账号获取API密钥
2. 在 .env 文件中取消注释DeepSeek配置
3. 注释掉Ollama配置

选项3: 使用Anthropic官方API
--------------------------------
1. 获取 Anthropic API 密钥
2. 在 .env 文件中取消注释Anthropic配置
3. 注释掉OpenAI兼容API配置

三、常见问题解决：

1. "'ull' 不是内部或外部命令"
   - 原因: Bun未正确安装或PATH问题
   - 解决: 重新安装Bun或检查PATH

2. "API Error: 403"
   - 原因: API密钥未配置或无效
   - 解决: 检查 .env 文件中的API配置

3. 启动后立即退出
   - 原因: 依赖未安装
   - 解决: 手动运行 bun install

四、手动启动命令：

1. 安装依赖: bun install
2. 启动TUI: bun run dev
3. 构建版本: bun run build

五、配置文件说明：

- .env.example: 配置示例文件
- .env: 实际使用的配置文件（从.example复制）
- TUI启动器.bat: Windows启动脚本

六、获取帮助：

1. 查看项目文档: README.md
2. 检查环境变量说明: ENVIRONMENT_VARIABLES.md
3. 查看项目说明: CLAUDE.md

注意：首次使用建议从Ollama开始，最简单易用。
