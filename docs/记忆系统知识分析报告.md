#### 2.2 Windows CMD窗口行为关键知识

#### 摘要
Windows命令行窗口生命周期管理的重要经验：双击BAT文件与手动打开CMD窗口的本质区别。

#### 核心发现
- 双击BAT文件创建的临时窗口会在脚本结束后自动关闭（系统设计限制）
- 手动打开CMD窗口创建的持久窗口由用户控制生命周期
- 无法通过修改BAT文件内容改变临时窗口的行为

#### 技术原理
- 临时窗口：`explorer.exe`调用`cmd.exe /c "your_script.bat"`
- 持久窗口：用户直接运行`cmd.exe`
